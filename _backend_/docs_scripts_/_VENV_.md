## Instalar o .venv

```{bash}
python -m venv .venv
```

## Ativar o .venv (Ambiente Virtual Python)

```{bash}
.venv\Scripts\activate
```

## Atualize o pip

```{bash}
python.exe -m pip install --upgrade pip
```

# Instale as dependências
```{bash}
pip install -r requirements.txt
```

## Salve as dependências

```{bash}
pip freeze > requirements.txt
```
